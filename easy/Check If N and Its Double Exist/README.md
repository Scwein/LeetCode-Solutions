# Решение задачи Check If N and Its Double Exist

## Описание задачи
Дан целочисленный массив `arr`. Необходимо определить, существуют ли в массиве два различных индекса `i` и `j`, такие что:
- `arr[i] == 2 * arr[j]`
- `i != j`

**Требования:**
- Вернуть `true`, если такая пара существует
- Вернуть `false`, если такой пары нет

## Обзор решения
Решение использует **хэш-таблицу** для отслеживания уже просмотренных чисел. Для каждого элемента проверяем, встречалось ли ранее его удвоенное значение или его половина (если число чётное).

## Алгоритм

1. **Инициализация**
   - Создаём пустую хэш-таблицу `$map` для хранения просмотренных чисел

2. **Проход по массиву**
   - Для каждого числа `$num` в массиве:
     - Проверяем, есть ли в `$map` число `$num * 2`
     - Проверяем, есть ли в `$map` число `$num / 2`, **только если `$num` чётное**
     - Если любое из условий истинно → возвращаем `true`
     - Иначе добавляем текущее число в `$map`

3. **Возврат результата**
   - Если цикл завершился без нахождения пары → возвращаем `false`
