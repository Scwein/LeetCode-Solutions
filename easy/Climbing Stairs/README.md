# Решение задачи Climbing Stairs

## Описание задачи
Вы поднимаетесь по лестнице. Чтобы достичь вершины, нужно сделать `n` шагов.

За один раз вы можете подняться либо на **1 ступеньку**, либо на **2 ступеньки**.

**Задача:** Сколько существует различных способов подняться на вершину?

## Обзор решения
Решение использует **динамическое программирование**, так как задача имеет оптимальную подструктуру и перекрывающиеся подзадачи.

Количество способов достичь `i`-й ступеньки равно сумме способов достичь `(i-1)`-й и `(i-2)`-й ступенек.
