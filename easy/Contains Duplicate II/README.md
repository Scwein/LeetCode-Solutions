# Решение задачи Contains Duplicate II

## Описание задачи
Дан целочисленный массив `nums` и целое число `k`. Необходимо определить, существуют ли в массиве два **различных индекса** `i` и `j`, такие что:
- `nums[i] == nums[j]`
- `abs(i - j) <= k`

**Требования:**
- Вернуть `true`, если такая пара существует
- Вернуть `false`, если такой пары нет

## Обзор решения
Решение использует **плавающее окно** фиксированного размера `k`. В окне хранятся уникальные элементы с помощью хэш-таблицы. При добавлении нового элемента проверяем, не встречался ли он уже в текущем окне.

## Алгоритм
1. **Инициализация**
   - Создаём пустую хэш-таблицу `$window` для хранения элементов в текущем окне

2. **Проход по массиву**
   - Для каждого индекса `i` от 0 до конца массива:
     - Если текущий элемент уже есть в `$window` → возвращаем `true` (нашли дубликат в пределах окна)
     - Добавляем текущий элемент в `$window`
     - Если размер окна превысил `k`, удаляем самый старый элемент (на индексе `i - k`)

3. **Возврат результата**
   - Если цикл завершился без нахождения дубликатов → возвращаем `false`
