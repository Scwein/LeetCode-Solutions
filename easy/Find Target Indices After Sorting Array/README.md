# Решение задачи Target Indices

## Описание задачи
Дан целочисленный массив `nums` и целое число `target`. Необходимо вернуть список индексов, на которых `target` находится в массиве **после сортировки** в порядке возрастания.

**Условия:**
- Массив может содержать повторяющиеся элементы
- Если `target` отсутствует в массиве, вернуть пустой массив
- Индексы должны быть отсортированы в порядке возрастания

## Обзор решения
Решение состоит из двух простых шагов:
1. Сортировка массива в порядке возрастания
2. Поиск всех индексов, содержащих `target`

## Алгоритм

1. **Сортировка массива**
   - Используем встроенную функцию `sort()` для сортировки по возрастанию

2. **Поиск целевых индексов**
   - Проходим по отсортированному массиву
   - Если текущий элемент равен `target`, добавляем его индекс в результат
   - Продолжаем до конца массива

3. **Возврат результата**
   - Возвращаем массив индексов
