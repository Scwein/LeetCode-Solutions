# Решение задачи Happy Number

## Описание задачи
Написать алгоритм, который определяет, является ли число `n` **счастливым**.

**Счастливое число** определяется следующим процессом:
1. Начиная с любого положительного целого числа, заменить число суммой квадратов его цифр
2. Повторять процесс до тех пор, пока число не станет равным 1, или пока не войдёт в бесконечный цикл
3. Число, для которого этот процесс заканчивается на 1, считается счастливым

## Обзор решения
Решение использует хэш-таблицу для отслеживания уже встреченных чисел. Если число встречается повторно, значит мы попали в цикл и число не может быть счастливым.

## Алгоритм
1. **Инициализация**
   - Создать пустой массив `$seen` для хранения уже встреченных чисел

2. **Бесконечный цикл**
   - Вычислить сумму квадратов цифр текущего числа `$n`
   - Если сумма равна 1 — вернуть `true` (число счастливое)
   - Если сумма уже встречалась ранее — вернуть `false` (обнаружен цикл)
   - Иначе:
     - Запомнить сумму как встреченное число
     - Присвоить `$n` новое значение суммы
     - Продолжить цикл

3. **Результат**
   - Функция возвращает `true`, если число счастливое, иначе `false`
