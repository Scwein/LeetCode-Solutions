# Решение задачи Intersection of Two Arrays

## Описание задачи
Даны два целочисленных массива `nums1` и `nums2`. Необходимо вернуть массив, содержащий **уникальные** элементы, которые присутствуют в обоих массивах.

**Результат может быть возвращён в любом порядке.**

## Обзор решения
Решение использует метод двух указателей после сортировки массивов. Это позволяет эффективно найти общие элементы без использования дополнительной памяти для хэш-таблиц.

## Алгоритм
1. **Сортировка массивов**
   - Сортируем оба массива в порядке возрастания
   - Это позволяет использовать два указателя для синхронного прохода

2. **Два указателя**
   - `p1` — указатель для `nums1`
   - `p2` — указатель для `nums2`
   - Проходим по обоим массивам одновременно

3. **Поиск пересечений**
   - Если `nums1[p1] == nums2[p2]`:
     - Добавляем элемент в результат
     - Сдвигаем оба указателя вперёд
   - Если `nums1[p1] < nums2[p2]`:
     - Сдвигаем `p1` вперёд
   - Если `nums1[p1] > nums2[p2]`:
     - Сдвигаем `p2` вперёд

4. **Удаление дубликатов**
   - Используем `array_unique()` для удаления возможных дубликатов в результате
