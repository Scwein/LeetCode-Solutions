# Решение задачи Majority Element

## Описание задачи
Дан массив `nums` размером `n`. Необходимо найти **элемент большинства** — элемент, который встречается более `⌊n / 2⌋` раз.

**Важно:**
- Можно предположить, что массив **не пуст** и элемент большинства **всегда существует**
- Элемент большинства встречается в массиве более чем `n/2` раз

## Обзор решения
Решение использует **сортировку** массива, после чего элемент большинства обязательно окажется в середине (или будет легко найден через подсчёт последовательных повторений). Алгоритм подсчитывает длину каждой группы одинаковых элементов и находит максимальную.

## Алгоритм

1. **Обработка частного случая**
   - Если в массиве один элемент, сразу возвращаем его

2. **Сортировка массива**
   - Сортируем массив в порядке возрастания
   - После сортировки одинаковые элементы группируются вместе

3. **Поиск самой длинной группы**
   - `$cnt` — длина текущей группы одинаковых элементов
   - `$maxCnt` — максимальная найденная длина
   - Проходим по массиву с 1-го индекса:
     - Если текущий элемент равен предыдущему — увеличиваем счётчик
     - Если не равен:
       - Проверяем, не стала ли текущая группа максимальной
       - Обновляем `$maxCnt` и запоминаем элемент
       - Сбрасываем счётчик на 1

4. **Проверка последней группы**
   - После цикла проверяем, не стала ли последняя группа максимальной
   - Возвращаем найденный элемент
