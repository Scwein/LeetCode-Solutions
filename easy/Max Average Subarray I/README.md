# Решение задачи Maximum Average Subarray I

## Описание задачи
Дан целочисленный массив `nums`, состоящий из `n` элементов, и целое число `k`. Необходимо найти **максимальное среднее значение** подмассива длины `k`.

**Требования:**
- Вернуть максимальное среднее арифметическое среди всех подмассивов длины `k`
- Результат должен быть вычислен с точностью до 10⁻⁵

## Обзор решения
Решение использует **плавающее окно** фиксированного размера `k`. Вместо вычисления суммы каждого подмассива с нуля, мы обновляем сумму окна, вычитая выходящий элемент и добавляя новый.

## Алгоритм

1. **Вычисление суммы первого окна**
   - Суммируем первые `k` элементов массива
   - Запоминаем эту сумму как текущий максимум

2. **Скольжение окна**
   - Для каждого индекса `r` от `k` до конца массива:
     - `l = r - k` — индекс выходящего элемента
     - Обновляем сумму окна: `windowAvgSum += nums[r] - nums[l]`
     - Если новая сумма больше максимальной, обновляем максимум

3. **Вычисление среднего**
   - Делим максимальную сумму на `k`
   - Возвращаем результат
