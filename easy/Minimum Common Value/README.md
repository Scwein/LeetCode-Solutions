# Решение задачи Minimum Common Value

## Описание задачи
Даны два целочисленных массива `nums1` и `nums2`, отсортированных в порядке возрастания.

**Задача:** Найти **минимальное общее значение** — наименьшее целое число, которое присутствует в обоих массивах.

**Условия:**
- Если общее значение найдено, вернуть его
- Если общих значений нет, вернуть -1

## Обзор решения
Решение использует метод **двух указателей** для эффективного поиска минимального общего элемента. Так как оба массива отсортированы, мы можем синхронно проходить по ним, сравнивая текущие элементы.

## Алгоритм

1. **Инициализация указателей**
   - `p1 = 0` — указатель для `nums1`
   - `p2 = 0` — указатель для `nums2`

2. **Синхронный проход**
   - Пока оба указателя не достигли концов массивов:
     - Если `nums1[p1] == nums2[p2]` → нашли общий элемент, возвращаем его
     - Если `nums1[p1] < nums2[p2]` → сдвигаем `p1` (элемент в первом массиве слишком маленький)
     - Если `nums1[p1] > nums2[p2]` → сдвигаем `p2` (элемент во втором массиве слишком маленький)

3. **Возврат результата**
   - Если цикл завершился, общих элементов нет → возвращаем -1
