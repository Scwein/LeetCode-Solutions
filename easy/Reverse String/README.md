# Решение задачи Reverse String

## Описание задачи
Дан массив символов `s`. Необходимо перевернуть строку, изменив порядок символов на противоположный.

**Важно:** операция должна выполняться **in-place** (без создания нового массива) с использованием O(1) дополнительной памяти.

## Обзор решения
Решение использует метод двух указателей для эффективного переворота массива символов за один проход, меняя местами символы с обоих концов.

## Алгоритм
1. **Инициализация указателей**
   - `p1` — указывает на начало массива (индекс 0)
   - `p2` — указывает на конец массива (индекс `count($s) - 1`)

2. **Обмен символов**
   - Пока `p1` меньше `p2 + 1` (то есть указатели не пересеклись):
     - Сохраняем символ из позиции `p1` во временную переменную
     - Записываем символ из позиции `p2` в позицию `p1`
     - Записываем сохранённый символ в позицию `p2`
     - Увеличиваем `p1` на 1
     - Уменьшаем `p2` на 1

3. **Результат**
   - Массив символов перевёрнут in-place
