# Решение задачи Search Insert Position

## Описание задачи
Дан отсортированный массив различных целых чисел `nums` и целевое значение `target`.

**Задача:** 
- Если `target` найден в массиве, вернуть его индекс
- Если `target` не найден, вернуть индекс, на котором он должен быть вставлен для сохранения порядка сортировки

**Требования:** Алгоритм должен иметь сложность O(log n).

## Обзор решения
Решение использует **бинарный поиск** для эффективного поиска позиции элемента. Если элемент не найден, бинарный поиск естественным образом определяет позицию для вставки.

## Алгоритм

1. **Инициализация границ**
   - `l = 0` — левая граница
   - `r = count($nums) - 1` — правая граница

2. **Бинарный поиск**
   - Пока `l <= r`:
     - Вычисляем середину: `m = l + (r - l) // 2`
     - Если `nums[m] == target` → возвращаем `m`
     - Если `nums[m] < target` → `l = m + 1`
     - Если `nums[m] > target` → `r = m - 1`

3. **Возврат позиции вставки**
   - Если цикл завершился, `target` не найден
   - Возвращаем `l` — это позиция, куда нужно вставить `target`
