# Решение задачи Sort Array By Parity

## Описание задачи
Дан целочисленный массив `nums`. Необходимо переставить элементы массива так, чтобы:
- **Все чётные числа** оказались в начале массива
- **Все нечётные числа** оказались в конце массива

**Важно:** Относительный порядок элементов внутри групп (чётных и нечётных) **не имеет значения**. Результат может быть в любом порядке, удовлетворяющем условию.

## Обзор решения
Решение использует метод двух указателей (slow и fast) для эффективной перестановки элементов за один проход.

## Алгоритм
1. **Инициализация указателей**
   - `slow` — указывает на границу между чётными и нечётными элементами
   - `fast` — используется для поиска чётных элементов

2. **Обход массива**
   - Проходим по массиву с помощью указателя `fast`
   - Если текущий элемент `nums[$fast]` является **чётным**:
     - Меняем местами элементы на позициях `slow` и `fast`
     - Увеличиваем `slow` на 1
   - Увеличиваем `fast` на 1 (на каждой итерации)

3. **Результат**
   - Все чётные элементы оказываются в начале массива
   - Все нечётные элементы оказываются в конце
