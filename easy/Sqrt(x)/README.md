# Решение задачи Sqrt(x)

## Описание задачи
Дано неотрицательное целое число `x`. Необходимо вычислить **целочисленный квадратный корень** — наибольшее целое число `y`, такое что `y * y ≤ x`.

**Требования:**
- Вернуть целую часть квадратного корня (округление вниз)
- Не использовать встроенные функции `sqrt()` или `pow()`
- Алгоритм должен быть эффективным

## Обзор решения
Решение использует **бинарный поиск** для нахождения наибольшего целого числа `m`, где `m * m ≤ x`. Бинарный поиск эффективно сужает диапазон возможных значений корня.

## Алгоритм

1. **Инициализация границ**
   - `l = 0` — левая граница
   - `r = x` — правая граница

2. **Бинарный поиск**
   - Пока `l <= r`:
     - Вычисляем середину: `m = l + (r - l) // 2`
     - Вычисляем квадрат: `square = m * m`
     - Если `square == x` → возвращаем `m`
     - Если `square < x` → `l = m + 1`
     - Если `square > x` → `r = m - 1`

3. **Возврат результата**
   - Если точный корень не найден, возвращаем `r`
   - `r` указывает на наибольшее число, квадрат которого ≤ x
