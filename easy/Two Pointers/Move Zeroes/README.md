# Решение задачи Move Zeroes

## Описание задачи
Дан целочисленный массив `nums`. Необходимо переместить все нули в конец массива, сохраняя относительный порядок ненулевых элементов.

**Важно:** операция должна выполняться **in-place** (без создания нового массива).

## Обзор решения
Решение использует метод двух указателей (slow и fast) для эффективного перемещения нулей в конец массива за один проход.

## Алгоритм
1. **Инициализация указателей**
   - `slow` — указывает на позицию, куда нужно поместить следующий ненулевой элемент
   - `fast` — используется для поиска ненулевых элементов

2. **Обход массива**
   - Проходим по массиву с помощью указателя `fast`
   - Если текущий элемент (`nums[$fast]`) не равен нулю:
     - Меняем местами элементы на позициях `slow` и `fast`
     - Увеличиваем `slow` на 1
   - Увеличиваем `fast` на 1 (на каждой итерации)

3. **Результат**
   - Все ненулевые элементы оказываются в начале массива в исходном порядке
   - Все нули автоматически перемещаются в конец
