# Решение задачи Valid Palindrome

## Описание задачи
Определить, является ли данная строка палиндромом, учитывая только буквенно-цифровые символы и игнорируя регистр.

## Обзор решения
Данное решение проверяет, является ли строка палиндромом, используя метод двух указателей после предварительной обработки строки — приведения к нижнему регистру и удаления всех символов, кроме букв и цифр.

## Алгоритм
1. **Предварительная обработка**
   - Привести всю строку к нижнему регистру для сравнения без учёта регистра
   - Удалить все символы, не являющиеся буквами или цифрами (спецсимволы, пробелы, знаки препинания)

2. **Сравнение двумя указателями**
   - Инициализировать два указателя: `p1` в начале (индекс 0) и `p2` в конце (индекс `len - 1`)
   - Сравнивать символы на позициях обоих указателей
   - Если символы не совпадают — вернуть `false`
   - Если символы совпадают — переместить `p1` вперёд, а `p2` назад
   - Продолжать, пока `p1` не встретится с `p2` или не пройдёт его

3. **Возврат результата**
   - Если все сравнения успешны — вернуть `true`

## Временная и пространственная сложность
- **Временная сложность:** O(n)
  - Приведение к нижнему регистру и удаление символов с помощью регулярного выражения занимает O(n)
  - Сравнение двумя указателями занимает O(n)

- **Пространственная сложность:** O(n)
  - Создаётся новая строка после удаления небуквенно-цифровых символов
