# Решение задачи Perfect Square

## Описание задачи
Дано целое положительное число `num`. Необходимо определить, является ли оно **полным квадратом**.

**полным квадрат** — это число, которое может быть выражено как квадрат некоторого целого числа.

## Обзор решения
Решение использует **бинарный поиск** для нахождения целого числа `x`, такого что `x * x = num`. Бинарный поиск эффективно сужает диапазон поиска, проверяя середину на каждой итерации.

## Алгоритм

1. **Обработка частного случая**
   - Если `num == 1`, сразу возвращаем `true` (1 = 1²)

2. **Инициализация границ**
   - `l = 1` — левая граница поиска
   - `r = num` — правая граница поиска

3. **Бинарный поиск**
   - Пока `r - l > 1` (между границами есть хотя бы одно число):
     - Вычисляем середину: `m = (l + r) // 2`
     - Вычисляем квадрат: `square = m * m`
     - Если `square == num` → возвращаем `true`
     - Если `square < num` → сдвигаем левую границу: `l = m`
     - Если `square > num` → сдвигаем правую границу: `r = m`

4. **Завершение**
   - Если число не найдено, возвращаем `false`
