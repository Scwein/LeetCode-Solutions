# Решение задачи Magical String

## Описание задачи
**Волшебная строка** `s` состоит только из символов `'1'` и `'2'` и подчиняется следующим правилам:
- Строка генерируется следующим образом:
  1. Начинается с `"122"`
  2. Каждая последующая группа символов определяется самой строкой
- Строка `s` является волшебной, если при её разбиении на группы последовательных одинаковых символов, длины этих групп образуют исходную строку

Более простое объяснение:
- Начинаем с `"122"`
- Читаем строку: `"1"`, `"22"` (группы символов)
- Это означает: одна `"1"`, затем две `"2"` → получаем `"1" + "22"` = `"122"` (начальная строка)
- Далее: следующая цифра в строке говорит, сколько символов добавить, а чередование говорит, какие символы добавлять

**Задача:** Для заданного `n` вернуть количество единиц (`'1'`) в первых `n` символах волшебной строки.

## Обзор решения
Решение состоит из двух частей:
1. Построение волшебной строки до длины `n` с помощью генерации
2. Подсчёт количества символов `'1'` в первых `n` символах

## Алгоритм

### Построение волшебной строки (`build_magical_str`)
1. **Инициализация:** Начинаем с базовой строки `"122"`
2. **Генерация:** Для каждого индекса `i` от 2 до `n-1`:
   - Смотрим на значение `s[i]` (оно указывает, сколько символов добавить: 1 или 2)
   - Смотрим на последний символ текущей строки (`s[-1]`)
   - Добавляем соответствующее количество противоположных символов:
     - Если добавляем 1 символ: добавляем `'1'` если последний был `'2'`, или `'2'` если последний был `'1'`
     - Если добавляем 2 символа: добавляем два одинаковых символа, противоположных последнему
3. **Завершение:** Возвращаем строку

### Подсчёт единиц
1. Проходим по первым `n` символам строки
2. Подсчитываем количество символов, равных `'1'`
3. Возвращаем полученное значение
