# Решение задачи Reverse Words in a String

## Описание задачи
Дана строка `s`, содержащая слова и пробелы. Необходимо перевернуть порядок слов в строке, сохраняя при этом порядок букв внутри каждого слова.

**Требования:**
- Удалить лишние пробелы (начальные, конечные, множественные пробелы между словами)
- Вернуть строку, в которой слова расположены в обратном порядке
- Между словами должен быть ровно один пробел

## Обзор решения
Решение использует простой подход:
1. Удаляет лишние пробелы с начала и конца строки
2. Разбивает строку на массив слов по пробелам
3. Проходит по массиву в обратном порядке и собирает результат
4. Удаляет последний лишний пробел

## Алгоритм
1. **Очистка строки**
   - `trim($s)` — удаляет пробелы в начале и конце строки

2. **Разбиение на слова**
   - `explode(' ', $s)` — разбивает строку на массив, используя пробел как разделитель
   - В результате могут появиться пустые элементы (при множественных пробелах)

3. **Обратный обход**
   - Проходим по массиву от последнего элемента к первому
   - Пропускаем пустые элементы (возникают из-за лишних пробелов)
   - Добавляем каждое слово к результату с пробелом

4. **Форматирование результата**
   - `rtrim($res)` — удаляет последний пробел в конце строки
