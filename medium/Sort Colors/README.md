# Решение задачи Sort Colors

## Описание задачи
Дан массив `nums`, содержащий целые числа 0, 1 и 2, представляющие красный, белый и синий цвета соответственно. Необходимо отсортировать массив **in-place** так, чтобы все элементы одного цвета были сгруппированы вместе в порядке: 0, 1, 2.

**Важно:** Решите задачу без использования встроенной функции сортировки.

**Цвета:**
- 0 — красный (red)
- 1 — белый (white)
- 2 — синий (blue)

## Алгоритм

### Три указателя
- **`low`** — граница области с нулями (все элементы до `low` — это 0)
- **`mid`** — текущий элемент для обработки
- **`high`** — граница области с двойками (все элементы после `high` — это 2)

### Логика обработки
1. **Если `nums[$mid] == 0`**:
   - Меняем местами с элементом на позиции `low`
   - Увеличиваем `low` и `mid`

2. **Если `nums[$mid] == 1`**:
   - Просто увеличиваем `mid` (1 уже на правильном месте)

3. **Если `nums[$mid] == 2`**:
   - Меняем местами с элементом на позиции `high`
   - Уменьшаем `high`
   - **Не увеличиваем `mid`** (потому что пришедший элемент нужно проверить)
